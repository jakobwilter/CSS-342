PROJECT TITLE: Queues & LongInts
PURPOSE OF PROJECT: Implements data structures, constructors and basic functionality for queues.
VERSION or DATE: 2018
AUTHORS: Jakob Wilter

PROJECT DESCRIPTION: deque.cpp.h contains logic and functions for shifting, removing, or inserting nodes within a queue. A driver is provided for evaluating changes in a queue. longint.cpp condenses long integers by removing any unnecessary zeros from numbers inputted. A driver is provided for testing cases of long integers.
